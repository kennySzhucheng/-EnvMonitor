# 🔌 EnvMonitor 原理图接线手册

> 按组逐行连线。已标出每个元件的引脚。用 W 键画线。

---

## 第1组：电源

```
+5V 标签 ──────── C21 (正极 +)
C21 (负极 -) ──── GND
+3.3V 标签 ────── C22 (正极 +)
C22 (负极 -) ──── GND
然后把画面上所有的 GND 符号连在一起！
```

---

## 第2组：I²C上拉 + 模块接口

### 2a. 上拉电阻
```
+3.3V ─── R4(上端)       R4(下端) ─── J3 的 pin2 (SCL)
+3.3V ─── R5(上端)       R5(下端) ─── J3 的 pin3 (SDA)
+3.3V ─── R6(上端)       R6(下端) ─── J2 的 pin2 (DATA)
```

### 2b. 模块供电
```
+3.3V ─── J2 的 pin1
+3.3V ─── J3 的 pin1
 GND  ─── J2 的 pin4
 GND  ─── J3 的 pin4
```

---

## 第3组：HC-SR04 超声波 + ESP8266

### 3a. HC-SR04 (J4)
```
+5V ─── J4 的 pin1 (VCC)
GND ─── J4 的 pin4 (GND)
```
> pin2(Trig)和 pin3(Echo)的信号线等核心板那组时再连

### 3b. ESP8266-01S (J5)
```
+3.3V ─── J5 的 pin1 (VCC)
+3.3V ─── J5 的 pin3 (CH_PD/EN, 必须拉高才能工作)
 GND  ─── J5 的 pin8 (GND)
```
> pin5(TX)和 pin6(RX)等核心板那组时连

---

## 第4组：按键 (3路完全一样, 做一路后复制)

### KEY1 通道
```
+3.3V ─── R7(上端)
R7(下端) ─┬─ SW2(上端)     SW2(下端) ─── GND
           └─ C18(上端)    C18(下端) ─── GND
```
> R7下端分叉接SW2和C18的交点，拉出标签 "PA4_KEY1"

### KEY2 通道
```
+3.3V ─── R8(上端)
R8(下端) ─┬─ SW3(上端)     SW3(下端) ─── GND
           └─ C19(上端)    C19(下端) ─── GND
```
> R8下端 → 标签 "PA5_KEY2"

### KEY3 通道
```
+3.3V ─── R9(上端)
R9(下端) ─┬─ SW4(上端)     SW4(下端) ─── GND
           └─ C20(上端)    C20(下端) ─── GND
```
> R9下端 → 标签 "PA6_KEY3"

---

## 第5组：LED指示灯 (3路一样)

```
标签"PB12_LEDR" → R10(上端) → R10(下端) → D2(正极/三角那边) → D2(负极) → GND
标签"PB13_LEDY" → R11(上端) → R11(下端) → D3(正极/三角那边) → D3(负极) → GND
标签"PB14_LEDG" → R12(上端) → R12(下端) → D4(正极/三角那边) → D4(负极) → GND
```

---

## 第6组：蜂鸣器驱动

```
标签"PB15_BUZZ" → R13(左端) → R13(右端) → Q1 的 B (基极, 中间脚)
                                            └── R14(上端)
                                                R14(下端) → GND

+5V ─── D5(负极, 有横线那端)
         │
         └── Q1 的 C (集电极, 一般是上脚)
              │
              └── LS1 的 pin1 (+)

Q1 的 E (发射极, 下脚) → GND
LS1 的 pin2 (-) → GND
```

---

## 第7组：核心板插座 (最后连!)

把所有标签连到核心板对应的引脚：

| 标签名 | 连到核心板引脚 |
|--------|---------------|
| PA0_DHT22 | H1/H2 的 PA0 |
| PB6_SCL | H1/H2 的 PB6 |
| PB7_SDA | H1/H2 的 PB7 |
| PB0_TRIG | H1/H2 的 PB0 |
| PB1_ECHO | H1/H2 的 PB1 |
| PA9_TX | H1/H2 的 PA9 |
| PA10_RX | H1/H2 的 PA10 |
| PA4_KEY1 | H1/H2 的 PA4 |
| PA5_KEY2 | H1/H2 的 PA5 |
| PA6_KEY3 | H1/H2 的 PA6 |
| PB12_LEDR | H1/H2 的 PB12 |
| PB13_LEDY | H1/H2 的 PB13 |
| PB14_LEDG | H1/H2 的 PB14 |
| PB15_BUZZ | H1/H2 的 PB15 |
| +5V | H1/H2 的 5V 引脚 |
| +3.3V | H1/H2 的 3.3V 引脚 |
| GND | H1/H2 的 GND 引脚 |

## ⚡ 小技巧

- **W键** = 开始画线, 点起点 → 点终点
- **L键** = 添加标签, 输入标签名
- **Delete** = 删除选中的东西
- **Ctrl+Z** = 撤销
- 画完一组按 **Ctrl+S** 保存
