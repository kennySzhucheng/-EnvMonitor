## 第0章：PCB 是什么？

PCB（Printed Circuit Board）= 一块覆了铜的玻璃纤维板。铜被蚀刻成细细的"导线"（Track），把元件的引脚连接起来，替代面包板上杂乱的飞线。

```
面包板：  元件脚插在孔里 → 用杜邦线手动连接
PCB：     元件脚焊在焊盘上 → 铜箔蚀刻的细线自动连通
```

**你的 EnvMonitor PCB 是一块双面板（2层）：**
- **F.Cu（顶层）** — 正面铜皮，主要放元件和走线
- **B.Cu（底层）** — 背面铜皮，辅助走线 + 大面积 GND 铺铜

## 第1章：PCB 编辑器的核心概念

### 1.1 元件 = 封装（Footprint）= 实物在板上的"脚印"

| 原理图中叫 | PCB 中叫 | 就是 |
|-----------|---------|------|
| 元件 (Symbol) | 封装 (Footprint) | 实物焊在板上的铜皮图案 |
| 电阻 R4 | `R_Axial_DIN0207_...` | 两个焊盘相距 2.54mm |
| 按键 SW2 | `SW_PUSH_6mm` | 4 个焊盘 6×6mm 排列 |

### 1.2 焊盘（Pad）= 元件脚焊上去的铜片

每个封装由若干个焊盘组成。焊盘有两种：
- **THT（通孔）焊盘**：中间有孔，元件脚穿过去在背面焊接
- **SMD（贴片）焊盘**：表面铜片，元件贴在表面焊接

你的板子全是 THT，意味着每个焊盘都有孔。

### 1.3 走线（Track）= 连接焊盘的铜线

在铜层上画一条线，电流就从 A 焊盘流到 B 焊盘。

### 1.4 飞线（Ratsnest）= "你应该这样连"

那些白色的细线就是飞线——它们不是真实的铜，只是提示："这两个焊盘是同一个网络，你应该画铜线连起来。"

> 飞线越少 = 布线越接近完成

### 1.5 过孔（Via）= 穿越板层的隧道

当一根线在顶层走到一半被挡住了，可以打一个过孔"钻"到底层继续走。

### 1.6 铺铜（Zone / Copper Pour）= 大面积铜皮

在板子的空白区域铺上一整块铜皮（通常是 GND 地），好处：
- 降低 GND 电阻，提供稳定参考地
- 散热更好
- 厂家蚀刻掉的铜更少（环保+省钱）

### 1.7 丝印（Silkscreen）= 板上的白字

标注元件编号（R4、C18）和方向标记（正极、pin1），方便焊接和调试。

## 第2章：常用操作速查

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 移动元件 | `M` | 选中后移到新位置 |
| 旋转元件 | `R` | 每次旋转 90° |
| 拖动元件 | `G` | 元件移动但连好的线跟着走 |
| 画走线 | `X` | 开始布线，点击起点 → 点终点 |
| 打过孔 | `V` | 布线过程中按 V 切换到底层 |
| 删除走线 | `Delete` | 选中线后删除 |
| 切换层 | `Ctrl+Tab` 或点击面板 | 在 F.Cu / B.Cu 之间切换 |
| 铺铜 | `B` | 重新填充所有铺铜区域 |
| 测量距离 | `Ctrl+Shift+M` | 测量两点之间距离 |
| DRC 检查 | 顶部工具栏 DRC 图标 | 设计规则检查 |

## 第3章：布局的黄金法则

### 法则1：按信号流摆放

信号从左到右、从上到下流动。不要东一个元件西一个元件跳来跳去。

```
❌ 坏的布局：                  ✅ 好的布局：

[传感器]──20cm──[MCU]          [传感器]  [MCU]
    │                         (信号线短)
    └──15cm──[电阻]               │
                   │           [电阻]
                   └──10cm──[LED]  │
                                [LED]
```

### 法则2：核心板居中

H1/H2（Blue Pill 排母）放在板子中央偏上的位置。因为所有信号都从核心板发散出去，它应该像"枢纽站"一样在中心。

### 法则3：模块接口放板子边缘

J2(DHT22)、J3(OLED)、J4(HC-SR04)、J5(ESP8266) —— 这些是外部模块的插座，必须放在板边，方便插拔。

### 法则4：同类元件对齐

电阻和电阻排成一行、LED 和 LED 排成一行，间距一致。一是好看，二是方便焊接。

### 法则5：留出焊接空间

THT 元件之间至少留 **2-3mm** 间距。太挤的话烙铁头伸不进去，维修也困难。

### 法则6：先大后小

先放大的元件（排母 H1/H2、电解电容 C21/C22、模块插座 J2-J5），再放小的（电阻、LED、按键）。大元件决定板子布局的大框架。

## 第4章：布线的黄金法则

### 法则1：短、直、少过孔

线越短越好，拐弯越少越好。THT 板尽量全在底层走线，顶层留给元件。

### 法则2：电源线要粗

| 线宽 | 适合走什么 | 承载电流 |
|------|----------|---------|
| 0.3mm | 信号线（GPIO、I²C、UART） | ~0.5A |
| 0.5mm | 一般电源线 | ~1A |
| 0.8mm-1.0mm | **+5V 和 +3.3V 主干道** | ~2A |

> 你先用 0.3mm 走信号线，0.8mm 走电源主干线。GND 不用特意走线——最后铺铜一步解决。

### 法则3：走线不要锐角转弯

```
❌ 锐角：         ❌ 直角：        ✅ 45°角：
   \               │                ───┐
    \              │                   │
     \             └───                └──
```

KiCad 布线时默认就是 45° 角，保持默认即可。

### 法则4：先布信号线，再布电源线，最后 GND 铺铜

布线顺序：
1. 先走最短、最简单的信号线
2. 再走电源线（比信号线粗）
3. 最后大面积 GND 铺铜覆盖所有空白区域

### 法则5：GND 靠铺铜，不靠走线

不要去画 GND 走线！最后在整块板上画一个 GND 铺铜区域，所有 GND 焊盘自动连通。这样既省事效果又好。

## 第5章：⚠️ 新手最容易犯的错误

| 排名 | 错误 | 后果 | 怎么避免 |
|------|------|------|---------|
| 1 | 忘记画板框 | 厂家不知道板子多大 | Edge.Cuts 层画个闭合矩形 |
| 2 | 走线和焊盘搭不上 | 电路断路 | 走线必须精确落在焊盘中心 |
| 3 | 顶层底层搞混 | 元件面走线被元件压住 | THT 走线全走 B.Cu（底层） |
| 4 | 线太细走电源 | 铜线发热或烧断 | +5V/+3.3V 用 0.8mm 以上 |
| 5 | 忘记铺铜 | GND 回路又长又细 | 布线完成后 B → 填充所有铺铜 |
| 6 | 铺铜后不跑 DRC | 短路或间距不够 | DRC → 全部通过才算完 |
| 7 | 元件摆到板框外面 | 厂家切掉了 | 所有元件必须在板框内 |

## 第6章：EnvMonitor 项目专属布局方案

### 6.1 板子尺寸

**建议：90mm × 65mm**（比之前说的 80×60 稍大，给手工焊接留余量）

> 嘉立创 10×10cm 以内打样 5 元/5片，你的板子完全在这个范围内。

### 6.2 推荐布局图

```

│  90mm                                               │
│  ┌──────────────────────────────────────────────┐   │
│  │  J3(OLED+BH1750)    J2(DHT22)                │   │
│  │  4P排母上边           4P排母右边              │   │
│  │                                              │   │
│  │  ┌─────────┐   ┌─────────┐                   │   │
│  │  │ H1 左排母│   │ H2 右排母│   ← Blue Pill    │   │
│  │  │ 2x20P   │   │ 2x20P   │     核心板插座     │   │
│  │  └─────────┘   └─────────┘                   │   │
│  │                                              │   │
│  │  R4 R5 R6   ← I²C+DHT22 上拉电阻             │   │

└─────────────────────────────────────────────────────┘
```

### 6.3 各区域详解

**区域A — 核心板（板子中上部）：**
- H1、H2 两个 2x20P 排母平行放置，间距与 Blue Pill 核心板的排针间距一致
- Blue Pill 标准排针间距为 **22.86mm（900mil）**，这个数据必须在布局时精确设置
- 核心板 USB 口朝上（板边缘方向），方便插 USB 线

**区域B — I²C 上拉电阻（核心板旁边）：**
- R4(4.7k)、R5(4.7k)、R6(4.7k) 三个电阻紧挨 H1/H2 的对应引脚
- R4 靠近 PB6(SCL)，R5 靠近 PB7(SDA)，R6 靠近 PA0(DHT22)

**区域C — 按键区（板子中下部）：**
- SW2、SW3、SW4 三个按键水平排成一行，间距 **10mm**（手指好按）
- 每个按键上方放对应的上拉电阻（R7/R8/R9）
- 每个按键旁边紧贴消抖电容（C18/C19/C20）
- 从右到左排列：KEY1(SW2)、KEY2(SW3)、KEY3(SW4)

**区域D — LED 指示灯（按键下方）：**
- D2(红)、D3(黄)、D4(绿) 水平排成一行，间距 **5mm**
- 每个 LED 旁边放限流电阻（R10/R11/R12）
- LED 朝向板边，方便从外壳窗口看到

**区域E — 蜂鸣器驱动（板子右下角）：**
- Q1(S8050)、R13(1k)、R14(10k)、D5(1N4148)、LS1(蜂鸣器)
- 这些元件紧密排列在一起，因为它们是同一个功能电路

**区域F — 模块接口（板子边缘）：**
- J2(DHT22)、J3(OLED+BH1750) 放上边缘
- J4(HC-SR04) 放右边
- J5(ESP8266) 放下边
- 所有模块接口的开口方向朝板外

**区域G — 电源滤波（任意空位）：**
- C21(100µF/16V) 靠近 +5V 来源处
- C22(100µF/10V) 靠近 +3.3V 来源处

## 第7章：布线的具体参数

### 7.1 设计规则设置

在 KiCad PCB 编辑器中：**File → Board Setup → Design Rules → Constraints**

| 规则 | 推荐值 | 说明 |
|------|--------|------|
| 最小线宽 | 0.3mm | 信号线默认宽度 |
| 最小间距 | 0.2mm | 铜线之间最小距离 |
| 最小过孔外径 | 0.8mm | 过孔的铜环外径 |
| 最小过孔钻孔 | 0.4mm | 过孔中间的孔 |

### 7.2 网络分类建议

在 **Board Setup → Net Classes** 中添加两个网络类：

| 网络类 | 线宽 | 包含哪些网络 |
|--------|------|------------|
| Power | 0.8mm | +5V, +3.3V |
| Default | 0.3mm | 所有信号（KEY1, I2C_SCL, LED_R 等） |

> GND 不需要出现在这里——它靠铺铜解决。

### 7.3 布线流程

```
步骤1: 关键信号线先走（I²C SCL/SDA, DHT22 DATA, HC-SR04 TRIG/ECHO, UART TX/RX）
         → 这些线要短，远离干扰源
   
步骤2: 一般信号线（LED, 按键, 蜂鸣器, ESP 控制脚）
         → 优先级较低，稍微绕一下没关系
   
步骤3: 电源线（+5V 和 +3.3V 用 0.8mm 粗线）
         → 从核心板供电引脚出发，依次经过各个需要供电的元件
   
步骤4: 铺地铜（在 B.Cu 层画一个覆盖全板的 GND zone）
         → 所有 GND 焊盘自动连接到大面积铜皮
         → F.Cu 层也铺 GND，两层之间打过孔连接
   
步骤5: 跑 DRC → 修正所有报错
```

## 第8章：DRC 检查清单

布线完成后，运行 DRC（设计规则检查），确保以下项目全部通过：

- [ ] **Unconnected items = 0** — 没有未连接的飞线
- [ ] **Clearance violations = 0** — 没有铜皮短路
- [ ] **Track width violations = 0** — 线宽符合规则
- [ ] **Board outline closed** — 板框是闭合图形
- [ ] **Copper edge clearance OK** — 铜皮不超出板框
- [ ] **Silkscreen not on pads** — 丝印文字没盖在焊盘上

## 第9章：输出 Gerber 文件（给工厂的文件）

### 9.1 导出步骤

PCB 编辑器中：**File → Fabrication Outputs → Gerbers**

| 要勾选的层 | 对应的文件 |
|-----------|----------|
| F.Cu | 顶层铜皮 |
| B.Cu | 底层铜皮 |
| F.Silkscreen | 顶层丝印（白字） |
| B.Silkscreen | 底层丝印 |
| F.Mask | 顶层阻焊 |
| B.Mask | 底层阻焊 |
| Edge.Cuts | 板框 |

### 9.2 导出钻孔文件

**File → Fabrication Outputs → Drill Files**

### 9.3 发送给嘉立创

把 Gerber 文件夹打包成 `.zip`，上传到 https://jlcpcb.com 下单。

参数选择：
- 层数：2 层
- 板厚：1.6mm
- 铜厚：1oz
- 颜色：绿色（最便宜）
- 数量：5 片
