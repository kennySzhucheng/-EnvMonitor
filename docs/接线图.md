# 🔌 引脚分配表

## 总览

```
STM32F103C8T6       外设                 说明
────────────────────────────────────────────────
PA0                 DHT22 DATA          1-Wire 单总线
PA1                 (预留 ADC)           可用于电池电压检测
PA9  (USART1_TX)    ESP8266 RX          UART 发送→WiFi接收
PA10 (USART1_RX)    ESP8266 TX          UART 接收←WiFi发送
PA4                 KEY1                按键1（页面切换）
PA5                 KEY2                按键2（确认）
PA6                 KEY3                按键3（返回）
PB0                 HC-SR04 Trig        超声波触发信号
PB1                 HC-SR04 Echo        超声波回响捕获（TIM3_CH4）
PB6  (I2C1_SCL)     OLED SCL + BH1750 SCL  共用I2C总线时钟
PB7  (I2C1_SDA)     OLED SDA + BH1750 SDA  共用I2C总线数据
PB12                LED_RED             红色LED（高温报警）
PB13                LED_YELLOW          黄色LED（湿度低）
PB14                LED_GREEN           绿色LED（靠近检测）
PB15                BUZZER              有源蜂鸣器（PWM驱动）
PA2  (USART2_TX)    (预留) 调试串口
PA3  (USART2_RX)    (预留) 调试串口
PA13 (SWDIO)        ST-Link 调试口
PA14 (SWCLK)        ST-Link 调试口
```

## I2C 总线设备地址
| 设备 | 7位地址 | 8位地址 |
|------|---------|---------|
| OLED SSD1306 | 0x3C | 0x78 |
| BH1750 | 0x23 | 0x46 |

## 系统时钟配置
- 主频：72MHz (HSE 8MHz × PLL)
- APB1：36MHz
- APB2：72MHz
- SysTick：1ms (FreeRTOS 心跳)

## 电源
- STM32 核心供电：3.3V (AMS1117 稳压)
- HC-SR04 供电：5V (USB 直接取电)
- 蜂鸣器供电：5V
